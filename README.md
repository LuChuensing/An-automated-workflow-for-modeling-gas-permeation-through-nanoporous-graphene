# An-automated-workflow-for-modeling-gas-permeation-through-nanoporous-graphene
A Python-based pipeline to automate graphene nanopore construction, gas molecule insertion, and energy barrier calculation for molecular dynamics simulations.
Firstly you should install ASE and Lammps
```
pip install ase
conda install -c conda-forge lammps
```
Then you use
```
python master.py
```
